# Honor Your Life Posters

A digital platform for Browning High School students to create and share their Honor Your Life posters, inspired by OPI's Honor Your Life initiative.

## Overview

This platform allows 9-12 grade students and staff to:
- Submit their personal information through a simple form
- Generate a custom HTML poster
- View a gallery of all posters created
- Edit and customize their poster using HTML/CSS

## Features

- 📝 **Simple Form Submission**: Students enter their information without any coding
- 🎨 **Auto-Generated Posters**: Instant HTML poster creation
- 🖼️ **Gallery View**: See all posters in one place
- ✏️ **Editable HTML**: Advanced users can customize their poster code
- 💾 **Local Storage**: Posters saved in the browser (no account needed)
- 🔄 **Export**: Save and share posters as HTML files

## Getting Started

1. Open `index.html` in a web browser
2. Fill out the form with your information
3. Click "Create Poster" to generate your poster
4. View your poster and edit if desired
5. Visit the Gallery to see all student posters

## What to Include in Your Poster

- Picture/family picture(s)
- Indian Name (if you have one)
- Tribal Affiliation (if enrolled)
- Area you live
- Likes/Hobbies
- Goals
- Motto/Favorite quote
- Strengths (given by a classmate or colleague)

## Technical Stack

- HTML5
- CSS3
- Vanilla JavaScript
- Browser Local Storage (no backend required)

## File Structure

- `index.html` - Main form page
- `poster-template.html` - Poster display template
- `styles.css` - All styling
- `script.js` - Form handling and gallery logic
- `docs/` - Student and teacher guides
